# Загрузка на GitHub через GitHub Desktop

Если командная строка не работает, используй GitHub Desktop - это намного проще!

## Шаг 1: Установи GitHub Desktop

1. Скачай: https://desktop.github.com/
2. Установи программу
3. Войди в свой GitHub аккаунт

## Шаг 2: Добавь проект

1. Открой GitHub Desktop
2. Нажми **File** → **Add Local Repository**
3. Выбери папку проекта: `C:\Users\kken9\Desktop\проекты кода\ShiKaraKa`
4. Нажми **Add Repository**

## Шаг 3: Создай коммит

1. В левой панели увидишь список всех файлов
2. Убедись что все файлы отмечены галочками
3. Внизу слева в поле "Summary" напиши:
   ```
   Initial commit: ShiKaraKa Anime Portal
   ```
4. Нажми синюю кнопку **Commit to main**

## Шаг 4: Опубликуй на GitHub

1. Вверху нажми **Publish repository**
2. Убедись что:
   - Name: `shikaraka-anime-portal`
   - Description: "Modern anime portal with React + Supabase"
   - **Сними галочку** "Keep this code private" (если хочешь публичный репозиторий)
3. Нажми **Publish repository**

## Готово! ✅

Через несколько секунд твой код будет на GitHub:
https://github.com/geygey123123/shikaraka-anime-portal

## Проверка

1. Открой https://github.com/geygey123123/shikaraka-anime-portal
2. Убедись что видишь все файлы:

   - package.json
   - README.md
   - и т.д.
